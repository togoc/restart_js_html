元素选择器

1.通配选择器

选定所有元素

*{styleRules}

*{ color:red; }

2.类选择器

选定指定类元素标签元素

elements{styleRules}

div{ color:red; }

3.属性选择器

根据简单属性选择

div[attr]{styleRules}

div[id]{ color:red; }

4.属性选择器

根据简单属性的值选择

div[attr="text"][attr="text"]{styleRules}

div[id="demo"]{ color:red; } 多条件选择 input[type="text"][class="demo"]{ color:green; }

5.属性选择器

属性有多个值，其中一个为val的元素

div[class~="val"]{styleRules}

div[class~="val"]{ color:red; }

6.属性选择器

属性有单个值，以val开头的元素

div[class^="val"]{styleRules}

div[class^="val"]{ color:red; }

7.属性选择器

属性有单个值，以val结尾的元素

div[class$="val"]{styleRules}

div[class$="val"]{ color:red; }

8.相邻选择器

元素对象相邻下一个元素

h1+p{styleRules}

h1+p{ color:red; }

9.兄弟选择器

元素之后所有满足条件的元素

h1~p{styleRules}

h1~p{ color:red; }

10.除xx之外选择器

多个元素中除某个元素外的元素

div:not(xx){styleRules}

h1~p{ color:red; }
